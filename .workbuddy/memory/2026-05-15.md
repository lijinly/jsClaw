## Session 持久化路径

```
{workspace-path}/.workspace/sessions/ws-{workspaceId}-s-{sessionId}.json
```

- `Session.save()` → 同步写入 JSON，自动创建父目录
- `Session.fromFile()` → 静态方法重建实例（workspace/member 引用由 caller 填充）
- `Workspace._restoreSessions()` → 启动时扫描 sessionsDir 恢复已有 Session
- `Zone.closeWorkspace()` → 调用 `workspace.save()` 保存所有 Session
