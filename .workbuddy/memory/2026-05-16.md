### 完整会话生命周期

```
启动
  startService()
    → Zone.loadWorkspace('default')
      → Workspace.initialize()
        → _restoreSessions()     ← 从磁盘恢复已有会话
          ↓
会话中
  chat(message, { sessionId })
    → session.userMessage()
      → this.save()              ← 每次消息后自动保存
          ↓
退出
  Ctrl+C / SIGTERM / SIGINT / exit
    → gracefulShutdown()
      → workspace.save()         ← 保存所有 Session
        → process.exit()
```
