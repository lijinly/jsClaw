# Weather Skill

This skill provides weather information for any city worldwide.

## Usage

When user asks about weather for a specific city, follow these steps:

1. Call `web_search` tool with a query like "weather in Beijing"
2. Extract the temperature and conditions from search results
3. Format the response in a friendly way

## Example

User: "What is the weather like in Shanghai?"

Response: "The current weather in Shanghai is partly cloudy with a temperature of 22°„C, humidity at 65%."
